This repository documents my journey through the **Python for Everybody (PY4E)** curriculum. It contains a comprehensive collection of theory notes, automated quiz summaries, and practical coding exercises designed to build a rigorous foundation for advanced data processing and linguistic analysis.

* **Institution:** Université Blida 1
* **Program:** Master's in Data Science & Natural Language Processing (DS & NLP)
* **Goal:** Mastering Python fundamentals to handle complex text corpora, algorithmic efficiency, and data-driven decision-making.

## 📂 Repository Structure

The repository is organized by lesson modules, each containing theory, assessment, and practice:

* **Lesson_0_Introduction:** Initial environment setup and the "first program" execution.
* **Lesson_1_The_Programmers_Mindset:** Understanding hardware architecture and the three pillars of program flow.
* **Lesson_2_Variables_Expressions_and_Statements:** Mastering data types, constants, and the mathematical order of evaluation.
* **Lesson_3_Conditional_Execution:** Implementing logic gates and robust error handling.
* **Lesson_4_Functions:** Building modular, reusable code blocks.
* **Lesson_5_Loops_and_Iteration:** Utilizing repetitive patterns for data aggregation and analysis.

## 🛠️ Key Skills Mastered

### 1. Hardware Architecture & Mindset
* Identified the roles of the **CPU** (The Focused Worker), **RAM** (Short-term Memory), and **Secondary Memory** (The Library).
* Mastered the three fundamental flow patterns: **Sequential**, **Conditional**, and **Repeated**.

### 2. Logic & Error Handling
* Implemented multi-way decision structures using `if`, `elif`, and `else`.
* Utilized the **Safety Net (`try/except`)** to prevent program crashes during invalid data input.
* Applied the **Guardian Pattern** to handle potential logic failures, such as preventing division by zero using short-circuiting: `$x>=2$ and $y!=0$ and $(x/y)>2$`.

### 3. Functional Programming
* Differentiated between **Arguments** (values passed) and **Parameters** (variable handles inside the function).
* Mastered **Fruitful Functions** that use the `return` keyword to send data back to the main program, as opposed to **Void Functions** which merely perform actions like printing.

### 4. Iteration Idioms
* Mastered **Indefinite Loops (`while`)** for continuous data collection and **Definite Loops (`for`)** for iterating through sequences.
* Implemented advanced loop control using `break` (The Emergency Exit) and `continue` (The Skip Button).
* Built robust algorithms for **Counting**, **Summing**, and finding **Maximum/Minimum** values using `None` as a sentinel value.
